how to add a new category ✦

1
Add a nav button. In the <nav>, copy a .nav-pill button, update its data-cat, label, and filter() argument to your new category name.
2
Add a color. In :root, add a CSS variable like --travel: #f59e0b; and add a matching .tag-travel { background: var(--travel); } rule.
3
Copy a section block. Duplicate any <div class="cat-section">, update its data-cat, the heading emoji + text, and the dot color.
4
Add post rows. Inside the new section's .posts div, copy a .post-row link and update the emoji, title, tag class, date, and read time. Done!
5
Optional featured post. Copy the .post-featured block from the Sports section if you want a big highlighted card at the top of your new category.
